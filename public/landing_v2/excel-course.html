The Complete 12-Day Excel Curriculum

Every Excel skill a professional Data Analyst is expected to know — in logical, practical sequence.

Phase 1 Foundations — Days 1–3 📗 sales_data.xlsx
  • Day 01
    Excel Orientation & Essential Formulas
    Ribbon, name box, formula bar · Relative ($A1), absolute ($A$1) & mixed references · SUM, AVERAGE, COUNT, COUNTA, MAX, MIN · Error types: #DIV/0!, #VALUE!, #REF!, #NAME?
    🎯 Calculate total sales, average order value, max single-day revenue
  • Day 02
    Formatting, Sorting & Filtering
    Number formats (currency, %, date, custom) · Conditional formatting: highlight cells, color scales, data bars, icon sets · Multi-level sort · AutoFilter & Advanced Filter
    🎯 Highlight top 10% of sales in green; filter for Q3 only
  • Day 03
    Data Cleaning Essentials
    TRIM, CLEAN · PROPER/UPPER/LOWER · Remove duplicates · Text to Columns · Find & Replace with wildcards · IFERROR/IFNA · Flash Fill (Ctrl+E)
    🎯 Clean a messy imported customer list (mixed case, extra spaces, duplicates)
Phase 2 Tables & Lookups — Days 4–6 📗 inventory_orders.xlsx
  • Day 04
    Excel Tables — The Right Way to Work
    Convert range → Table (Ctrl+T) · Structured references (Table1[Column]) · Auto-expand · Slicers · Total row
    🎯 Create a Table; add a total row; apply a Slicer for region
  • Day 05
    Lookup & Reference Functions
    VLOOKUP (exact + approximate match) · HLOOKUP · INDEX + MATCH combo · XLOOKUP (replaces both) · VLOOKUP limitations
    🎯 Use XLOOKUP to pull product prices from Products sheet into Orders sheet
  • Day 06
    Logic Functions
    IF, nested IFs · IFS (multi-condition) · AND, OR, NOT · IFERROR, IFNA · SWITCH for exact values
    🎯 Flag orders as 'Priority' if value > ₹10,000 AND customer is a repeat buyer
Phase 3 Data Manipulation — Days 7–8 📗 hr_data.xlsx
  • Day 07
    Text Functions
    LEFT, RIGHT, MID, LEN · FIND, SEARCH (case-sensitive vs not) · SUBSTITUTE · CONCATENATE / CONCAT / TEXTJOIN · TEXT(value, format)
    🎯 From "FirstName LastName" column, create separate first & last name columns
  • Day 08
    Date & Time Functions
    TODAY(), NOW() · DATE(), YEAR(), MONTH(), DAY(), WEEKDAY() · DATEDIF() · EDATE, EOMONTH · NETWORKDAYS
    🎯 Calculate employee tenure and flag who is eligible for 5-year bonus
Phase 4 Analysis Tools — Days 9–11 📗 sales_analysis.xlsx
  • Day 09
    Conditional Aggregation
    SUMIF / SUMIFS · COUNTIF / COUNTIFS · AVERAGEIF / AVERAGEIFS · MAXIFS / MINIFS · Wildcards in criteria ("*North*")
    🎯 Build a regional monthly revenue summary table using SUMIFS
  • Day 10
    PivotTables — Core Mechanics
    Insert → PivotTable · Rows, Columns, Values, Filters areas · Value field settings (Sum, Count, % of Total) · Group dates by Month/Quarter/Year
    🎯 Create a pivot showing monthly revenue by product category
  • Day 11
    PivotTables — Advanced + Charts
    Calculated Fields · Calculated Items · Slicers (visual filters) · Timeline for dates · PivotChart types & formatting · Refresh pivot
    🎯 Add "Profit Margin" calculated field; connect a Region Slicer
Capstone Real-World Tools & Dashboard Capstone — Day 12 📗 sales_analysis.xlsx
  • Day 12
    🏆 Data Validation, What-If Analysis & Final Dashboard
    Part A — Data Validation: Drop-down lists, number/date restrictions, custom formula validation, input messages, error alerts, sheet protection.

    Part B — What-If Analysis: Goal Seek, Scenario Manager (3 scenarios), 1-variable & 2-variable Data Tables.

    Part C — Capstone Dashboard: Data cleaning with TRIM + IFERROR · SUMIFS KPI tiles (revenue, orders, AOV) · PivotTable with MoM trend · Top-10 products bar chart · Slicer + Timeline · Conditional formatting heatmap · Goal Seek for annual target planning.
    🎯 Submit a fully interactive Sales Dashboard — your first portfolio piece
