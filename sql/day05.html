📊 Day 05 Executive Summary

Aggregations allow summarizing large datasets into key business metrics. Today you mastered the five core functions: COUNT, SUM, AVG, MIN, and MAX, along with the ROUND function for formatting, and string aggregation using STRING_AGG/GROUP_CONCAT.
